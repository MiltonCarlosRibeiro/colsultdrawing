Acesse:  pasta da sua aplicação:
cd C:\drawing-mapoteca


start node server.js


node server.js


Solução para rodar o node em backgroud mesmo se o cmd estiver fechado


npm install -g pm2

*********************
Agora rode o servidor com:
node server.js
pm2 start server.js

**************************

Para ver o satus

pm2 list

Para parar

 pm2 stop server.js

*********************************
